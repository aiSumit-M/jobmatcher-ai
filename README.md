# 🚀 JobMatcher – AI Powered Resume & JD Fit Analyzer

## 📌 Overview
JobMatcher is a smart web-based tool that analyzes resumes against job descriptions and generates a **Fit Score** to measure compatibility. It helps job seekers understand rejection reasons and improve their resumes.

---

## 🎯 Problem Statement
- Job seekers get rejected without feedback  
- Recruiters handle 200+ resumes per job  
- No simple, free, India-focused ATS tool exists  

---

## 💡 Solution
JobMatcher provides:
- ✅ Resume–JD Fit Score  
- ✅ Skill gap analysis  
- ✅ Keyword matching insights  
- ✅ Improvement suggestions  
- 🔒 100% privacy (client-side processing)  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- Vercel (Hosting)  

---

## ⚙️ How It Works
1. Upload Resume & Job Description  
2. Text parsing & cleaning  
3. Skill & keyword extraction  
4. Fit Score calculation  
5. Dashboard output  

---

## 📊 Fit Score Formula
```
Fit Score = 0.5 × Skill Match + 0.3 × Experience Match + 0.2 × Keyword Match
```
